# Scientific Calculator

A modern full-stack scientific calculator built with Java, Spring Boot, Thymeleaf,
Bootstrap, and JavaScript.

## Features
- Formula-based expression input
- Scientific expression evaluation
- Responsive web interface
- Spring Boot REST API backend
- Calculation history
- Dark mode
- Keyboard support
- Clean MVC architecture

## Technologies Used
- Java 21
- Spring Boot
- Spring MVC
- Thymeleaf
- Bootstrap 5
- JavaScript (ES6)
- EvalEx
- Maven

# Getting Started

## Prerequisites
- Java 21 or newer
- Maven

## Clone the Repository
```
git clone https://github.com/AidanNolan-maker/scientific-calculator.git
```

## Run the Application

On Windows:
```
mvnw.cmd spring-boot:run
```

On macOS/Linux:
```
./mvnw spring-boot:run
```
